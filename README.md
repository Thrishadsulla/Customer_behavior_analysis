# 🛒 Customer Shopping Behavior Analysis

An end-to-end **Data Analytics** project that analyzes customer shopping behavior using **Python, PostgreSQL, SQL, and Power BI**. The project demonstrates the complete analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL-based business analysis, interactive dashboard creation, and business recommendations. :contentReference[oaicite:0]{index=0}

---
# 📖 Project Overview
Understanding customer purchasing behavior is essential for improving sales strategies and customer satisfaction. This project analyzes **3,900 customer purchase records** to uncover insights into spending patterns, customer segments, product preferences, and subscription behavior. The findings help businesses make informed, data-driven decisions. :contentReference[oaicite:1]{index=1}

# 🔄 Project Workflow

## 1️⃣ Data Loading
- Imported dataset into Python using Pandas.
- Performed initial exploration using `info()` and `describe()`.

---
## 2️⃣ Data Cleaning

The dataset was cleaned before analysis by:
- Handling missing values
- Imputing missing review ratings
- Renaming columns using snake_case
- Creating new features (Age Group & Purchase Frequency)
- Removing redundant columns
- Loading cleaned data into PostgreSQL

## 4️⃣ SQL Business Analysis
The cleaned dataset was imported into PostgreSQL, and SQL queries were used to answer business questions.

### Business Questions Answered
- Revenue generated by gender
- High-spending customers using discounts
- Top-rated products
- Shipping type comparison
- Subscriber vs Non-subscriber spending
- Products most dependent on discounts
- Customer segmentation
- Top products in each category
- Repeat buyer analysis
- Revenue contribution by age group

# ▶️ How to Run

### Clone the Repository
```bash
git clone https://github.com/yourusername/Customer-Shopping-Behavior-Analysis.git
```

### Install Required Libraries
```bash
pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2
```

### Steps
1. Open the Jupyter Notebook.
2. Run all cells for data cleaning and EDA.
3. Load the cleaned dataset into PostgreSQL.
4. Execute the SQL queries.
5. Open the Power BI dashboard (`.pbix`) to explore the interactive visualizations.
